Dart 语言提高生产力

Dart 让你使用新的语言、类库和工具构建 Web 应用，如 Dart Editor IDE、包管理工具以及一个从 Dart 到 JavaScript 的编译器。Dart 带来了更高的性能和更好的项目伸缩性。

Polymer.dart 和 AngularDart

Polymer.dart 让你站在未来 Web 技术的基石 Web Components 之上，并且现在就可以用！深度结合 Dart 语言特性的 AngularDart 让你简单高效地从事 Web 开发。

生产环境中的 Dart

去年11月 Dart 发布了 1.0 版，现在你可以放心地将它用于生产环境的开发了。目前，Dart 已经被应用于很多实际的项目中。我们期待全世界的开发者使用 Dart 创造更多优秀应用！
